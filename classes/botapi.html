editMessage

  • editMessage(channelId: Snowflake, messageId: Snowflake, data: string | MessageEditData): Promise<Message>
    • Defined in src/structures/BotAPI.ts:277

    Edit a previously sent message. The fields content, embed and flags can be edited by the original message author. Other users can only edit flags and only if they have the Permission.ManageMessages permission in the corresponding channel.

    example
    message.edit('Updated content!');
    1. A MessageEditData object, containing any of the fields @example typescript message.edit({ content: 'Updated content!', embed: { title: 'My Embed!' } });

    Parameters

    • channelId: Snowflake

      The ID of the channel that contains the message you wish to edit

    • messageId: Snowflake

      The ID of the message you wish to edit

    • data: string | MessageEditData

      The updated message data. Can be:

      1. Raw content to be edited to

    Returns Promise<Message>

sendMessage

  • sendMessage(channelId: Snowflake, data: string | MessageData | MessageEmbed, options?: Partial<MessageOptions>): Promise<Message>
    • Defined in src/structures/BotAPI.ts:117

    Post a message to a GuildTextChannel or DMChannel. If operating on a GuildTextChannel, this method requires the Permission.SendMessages permission to be present on the current user. If the MessageOptions.tts field is set to true, the Permission.SendTTSMessages permission is required for the message to be spoken

    example
    channel.sendMessage('Hello World!');
    1. A MessageData object, containing content and/or embed @example typescript channel.sendMessage({ content: 'Hello World!', embed: { title: 'My Embed!' } });
    2. A MessageEmbed instance

    Parameters

    • channelId: Snowflake

      The ID of the channel to send the message in

    • data: string | MessageData | MessageEmbed

      The message data. Can be:

      1. Raw content to be sent as a message
    • Optional options: Partial<MessageOptions>

      The message's options

    Returns Promise<Message>
