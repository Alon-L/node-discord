shardChangedState

  • shardChangedState(state: BotSocketShardState, shardState: BotShardState, botEvent: BotStateEvents): void
    • Defined in src/socket/BotSocket.ts:115

    Called when a shard under this socket changes its state. If all shards under this socket now have the same state, a message will be sent to the BotShardManager telling it to emit an event for all shards

    example
    this.botSocket.shardChangedState(
 BotSocketShardState.Ready,
 BotShardState.Ready,
 BotEvents.Ready,
);

    Parameters

    • state: BotSocketShardState

      The state to be checked for

    • shardState: BotShardState

      The state BotShard should be at after sending the message

    • botEvent: BotStateEvents

      The event that should be emitted to all shards

    Returns void
